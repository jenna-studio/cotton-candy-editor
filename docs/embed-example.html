How to Embed

Include the Cotton Candy Editor in your project:

1. CDN (Recommended)

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/jenna-studio/cotton-candy-editor@main/dist/editor.min.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/gh/jenna-studio/cotton-candy-editor@main/dist/embed.min.js"></script>

2. Initialize the Editor

// Create container
<div id="my-editor"></div>

// Initialize editor
const editor = new CottonCandyEditor({
    parent: document.getElementById('my-editor'),
    theme: 'cotton-candy',
    language: 'javascript',
    value: 'console.log("Hello World!");'
});

3. API Methods

// Get/Set content
const code = editor.getValue();
editor.setValue('new code');

// Change theme/language
editor.setTheme('cotton-candy');
editor.setLanguage('python');

// Event handling
editor.on('change', (content) => {
    console.log('Code changed:', content);
});
